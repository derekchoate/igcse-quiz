The Secret Language
✦ 0 stars only ever go up

Every computer on Earth speaks one language.
Tonight, you learn it.

It has an alphabet of exactly two letters. Every game, every song, every message you've ever sent — all of it is written in this language. Most people never learn it. It takes about twenty minutes.

How this works: there are no marks, no timer, no wrong answers, and nothing to hand in. You flip switches and see what happens. Stop whenever you like, skip whatever you like, peek ahead if you're curious. Every discovery you make earns a star — and stars can never be taken away.

Deep inside every computer are billions of microscopic switches. Each one can only do two things: be on, or be off. That's it. That's the whole alphabet — on and off, written as 1 and 0.

Try it. Tap the bulb a few times and watch the digit underneath.

One switch can say two things. Which raises a question: what could two switches say…?

Give each switch a value — the right one is worth 1, the left one is worth 2. Light a switch and you add its value. The total is whatever's lit.

Can you make the total say…

There's no combination that doesn't work — with two switches there are only four ways to set them. Try all four and you'll find every number from 0 to 3 along the way.

Here's the pattern the whole language is built on: each new switch is worth double the one before. So four switches are worth 8, 4, 2 and 1 — and with just those four you can make every number from 0 to 15.

See if you can light up…

Start with the biggest switch that fits inside your number, then see what's left over. Going for 13? The 8 fits (13 − 8 leaves 5)… now what's the biggest switch that fits inside 5?
13 = 8 + 4 + 1. So: light 8, light 4, skip 2, light 1. In the secret language, thirteen is written 1101.

Eight switches together is called a byte — the basic unit of every file on every device you own. Eight switches can make any number from 0 all the way to 255.

Some numbers worth finding…

Same trick as before, just bigger: biggest switch that fits, then work on what's left. For 19: the 16 fits (leaving 3)… and you already know how to make 3 from the second discovery. For 255 — what happens if everything is lit?

Notice what just happened

Nobody stood at a whiteboard. Nobody set a deadline. Nobody marked anything. You poked at some switches, followed your curiosity, took a nudge when you wanted one — and now you can read and write binary, which is genuinely the foundation of computer science.

That's all self-study is. It's this, repeated. Some sessions will feel easy like tonight; some will be slower — and on the slow days you're allowed to stop, walk away, and come back. The stars don't expire. Neither does anything you've understood.
